## :비디오_게임: 게임 플레이 방법
1. 캐릭터 이동은 키보드 W, A, S, D키로 가능합니다.
2. 마우스 왼쪽 버튼를 누르면 참새를 던지고, 위험한 물체나 지형을 확인할 수 있습니다.
3. 죽으면 체크포인트에서 부활합니다.
4. 도넛을 먹으면 게임 클리어입니다!
